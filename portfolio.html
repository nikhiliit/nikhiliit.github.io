Sensor-to-Skeleton Diffusion Model

A novel approach for generating human skeleton frames from sensor data using diffusion models with Lipschitz regularization for improved human activity recognition.

Project Details

  • Collaboration: Cisco Research & Texas State University
  • Date: Nov 2023 - April 2024
  • Publication: MMM 2025

Technologies Used

  • PyTorch
  • Diffusion Models
  • Python
  • Lipschitz Regularization
  • Sensor Data Processing

Key Features

  • Novel sensor-to-skeleton diffusion approach
  • Improved stability with Lipschitz regularization
  • Enhanced human activity recognition accuracy
  • Low sensor data requirements
  • Real-time capable implementation
